CSS Selectors

CSS selectors are used to find HTML elements which we want to style. Selectors are part of CSS rule set. There are wide range of CSS selectors available, which allow us to precisely select elements that we want to style. CSS selectors select HTML elements according to its id, class, type, attribute etc. The elements which are selected by selector are referred to as subject of the selector.

There are several different types of selectors in CSS. Few basic and most used selectors are as follows:

  1. Element Selector
  2. Id Selector
  3. Class Selector
  4. Universal Selector
  5. Group Selector

ELEMENT SELECTOR: The element selector selects the HTML elements based upon the element name.

Syntax: elementname

Example: p{ } will match all the para elements in the HTML page and design them.

ID SELECTOR: The id selector selects an element based upon the value of its id attribute. The id should always be unique within the page, so that id selector can be used to uniquely select element. To select an element with specific id we use (#) character before the id of element.

Syntax: #idname

Example: #component1{ } will design HTML element with id=”component1”.

CLASS SELECTOR: The class selector selects all HTML elements that have the given class attribute. To select elements of specific class we use period (.) character before class name.

Syntax: classname

Example: .align{ } will design all HTML elements with class=”align”.

UNIVERSAL SELECTOR: The universal selector (*) selects all the HTML elements on the page.

Syntax: *

Example: *{ } will design and make changes to every element of the HTML page.

GROUP SELECTOR: The group selector selects all HTML elements with the same style definition. We use group selector to minimize the code. We use comma (,) to separate each selector.

Syntax: A, B, C

Example: h1, h2, p { } will provide same CSS properties to design all the elements.
